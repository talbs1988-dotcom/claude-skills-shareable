# 00 — Pre-flight Check

לפני שמתחילים בשאלון, צריך לוודא שיש למשתתפת את הבסיס. בלי זה — אין בוט.

## מה לבדוק (3 דברים)

### 1. מערכת הפעלה ומחשב פנוי

שאלי:

```
איזה מחשב יש לך — Mac או PC (Windows)?
ויש לך מקום פנוי במחשב לכ-200 מגה בייט של קבצים?
```

**אם Mac** → מסלול Mac אפשרי (גם הענן זמין)
**אם Windows** → מסלול Windows אפשרי (גם הענן זמין)
**אם אין מחשב או שהוא רק מהטלפון** → רק ענן (Railway)

### 2. מספר WhatsApp פעיל

שאלי:

```
איזה מספר WhatsApp את רוצה לחבר לבוט?
(זה המספר שתשלחי אליו הודעות והבוט יענה לך —
לרוב זה הטלפון האישי שלך)
```

הערה לעצמך: המספר הזה ייכנס ל-whitelist של הבוט. רק ממנו הבוט יקבל הודעות.

**אזהרה לתת לה:**

> 💡 חשוב לדעת: כשהבוט מחובר ל-WhatsApp שלך, את לא תוכל להשתמש ב-WhatsApp Web במקביל מאותו מספר. ה-WhatsApp Business במחשב יעבוד עדיין.

### 3. כרטיס אשראי לשירותים בתשלום

שאלי:

```
לבוט יש 3 שירותים חודשיים בתשלום (סה"כ כ-140₪ לחודש):

• Green API (חיבור לוואטסאפ) — $12 לחודש
• Claude Pro (המוח של הבוט) — $20 לחודש
• Railway (אירוח בענן, אם בחרת ענן) — $5 לחודש

יש לך כרטיס אשראי בינלאומי לרישום לשירותים האלה?
(או PayPal — Railway ו-Anthropic מקבלים גם)
```

**אם כן** → ממשיכים
**אם לא** → עוצרים. בלי תשלום, אי אפשר להמשיך. הצע לחזור כשיש לה.

---

## תוצרת השלב

לפני שאת ממשיכה לקטגוריה 1, צריך להיות לך תמונה ברורה של:

- מערכת ההפעלה (Mac / Windows / רק ענן)
- מספר WhatsApp הראשי שיחובר
- אישור על נכונות לעלויות החודשיות

**שמור אצלך:**

```
פרטי פתיחה:
- שם המשתתף: [שמה]
- מערכת הפעלה: [Mac/Windows/Cloud-only]
- WhatsApp: [מספר]
- אישור עלות: ✓
- תאריך התחלה: [היום]
```

זה ישמש אותך בכל הקטגוריות הבאות.

---

## אם משהו לא בסדר

- **אין לה Mac/PC ולא רוצה לשלם על ענן?** הסבר שאי-אפשר בלי לפחות אחד מהם. הצע שהוא תחזור כשיש לה.
- **חושש מהעלות?** הראי לה את החשבון: 140₪/חודש = 4.5₪/יום. סביר לבעלת עסק.
- **לא בטוח שזה לה?** הצע שתחשוב יום-יומיים ותחזור. אל תדחפי.

המטרה: שתתחיל את השאלון רק כשהוא **מוכן ובטוחה**. עדיף לעצור עכשיו מאשר לבנות בוט שלא משתמשים בו.
