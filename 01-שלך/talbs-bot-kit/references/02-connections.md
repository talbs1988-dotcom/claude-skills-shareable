# 02 — קטגוריה 2: חיבורים בסיסיים

המטרה: לפתוח את כל החשבונות החיצוניים שהבוט יחבר אליהם, ולקבל את ה-credentials.

## למה חשוב

הבוט לבד הוא רק קוד. הוא צריך 3 חיבורים כדי לעבוד:

- **Green API** — מתרגם בין WhatsApp לבוט
- **Anthropic** — המוח של הבוט (Claude)
- **Airtable** — ה-CRM (לידים, משימות) — נדבר עליו בקטגוריה 4, פה רק פותחים חשבון

## רצף הפעולות

עבור על 3 השירותים לפי הסדר. **אל תדלגי** — כל אחד מהם תלוי בקודם.

---

## 2.1 — Green API (חיבור ל-WhatsApp)

הסבר לה:

```
Green API מתחבר ל-WhatsApp שלך באמצעות סריקת QR code,
ומאפשר לבוט לקרוא/לשלוח הודעות.

עלות: $12 לחודש (לאחר ניסיון חינם של כמה ימים).
```

**שלבי ההרשמה:**

1. שלח לה את הקישור:
   ```
   https://console.green-api.com/
   ```
2. הנחיה:
   ```
   - לחצי על "Sign up"
   - הירשמי עם המייל שלך
   - לאחר הרשמה, תיכנס לדאשבורד וצרי instance חדש (תוכנית "Developer" או "Business")
   ```
3. כשהוא בדאשבורד — בקש שתעתיק 2 ערכים ותשלח לך:

   ```
   צריך ממך 2 דברים מתוך הדאשבורד של Green API:

   1. Instance ID (מספר של 10 ספרות, מתחיל בדרך כלל ב-71)
   2. API Token (מחרוזת ארוכה של אותיות ומספרים)

   זה שני שדות תחת ה-instance שיצרת.
   ```

4. שמור במשתנים:
   ```
   GREEN_API_INSTANCE_ID = "71XXXXXXXX"
   GREEN_API_TOKEN       = "xxxxxxxxxxxxxxxxxxxxxx"
   ```

**עוד לא לסרוק את ה-QR** — נעשה את זה בשלב הפריסה (אחרי שהבוט מותקן).

---

## 2.2 — Anthropic (המוח של הבוט)

הסבר:

```
Claude הוא ה-AI שמניע את הבוט. צריך מנוי Pro
(או API key, אבל Pro פשוט יותר ועולה פחות).

עלות: $20 לחודש.
```

**שלבי ההרשמה:**

1. שלח לה את הקישור:
   ```
   https://claude.ai/
   ```
2. הנחיה:
   ```
   - הירשמי עם המייל שלך
   - הצטרפי ל-Pro plan ($20/חודש)
   - לאחר התשלום, לכי ל-Settings → API Keys
   - צור API Key חדש (שם: "WhatsApp Bot")
   - העתק את ה-key (מתחיל ב-sk-ant-...)
   ```
3. שמור:
   ```
   ANTHROPIC_API_KEY = "sk-ant-xxxxxxxxxxxxxxxxxx"
   ```

**אזהרה חשובה לתת לה:**

> 🔐 ה-API key הזה זה כמו סיסמה. אל תשתף אותו עם אף אחד. אם הוא דולף — חזרי ל-Anthropic ומחקי אותו, צור חדש.

---

## 2.3 — Airtable (CRM — חשבון בלבד עכשיו)

הסבר:

```
Airtable זה כמו Excel חכם — שם הבוט שומר את הלידים, המשימות,
והעסקאות שלך. אצטריך את חשבון שלך, אבל את המבנה (טבלאות, שדות)
נקבע בקטגוריה 4 — נשכפל תבנית מוכן.

עלות: חינם (תוכנית Free מספיקה לבוט אישי).
```

**שלבי ההרשמה:**

1. שלח לה את הקישור:
   ```
   https://airtable.com/signup
   ```
2. הנחיה:
   ```
   - הירשמי עם המייל שלך
   - בחר תוכנית Free
   - בסיום ההרשמה — תגיד לי "פתחתי".
   - שמור את המייל שאיתו נרשמת — נצטרך אותו לשכפול התבנית בקטגוריה 4.
   ```
3. שמור:
   ```
   AIRTABLE_EMAIL = "user@example.com"
   ```

לא צריך API key כרגע. נקבל אותו אחרי שכפול ה-base.

## פלט הקטגוריה

לפני שעוברים לקטגוריה 3, צריכים להיות אצלך:

```
✅ Green API Instance ID:  71XXXXXXXX
✅ Green API Token:        xxxxxxxxxxxxxxxxx
✅ Anthropic API Key:      sk-ant-xxxxxxxxxxxx
✅ Airtable Email:         user@example.com
✅ WhatsApp Number:        972XXXXXXXXX  (מ-preflight)
```

**אל תמשיך בלי כל החמישה.** אם משהו חסר — חזרי וסיימי לפני קטגוריה 3.

---

## טיפים לדיאלוג

- **כל שירות לוקח 5-10 דקות הרשמה.** סך הכל ~30 דקות לקטגוריה הזו. תכיני אותה לזמן.
- **אם משהו נתקע** (הרשמה לא עוברת, אישור מייל לא מגיע) — הצע לעבור הלאה ולחזור בסוף. לא כדאי להיתקע.
- **כשהוא נותן לך credentials** — אל תחזור עליהם פומבית בהודעה. רק "קיבלתי, ממשיכים".
- **שמירת credentials**: עד שכותבים את ה-`.env` בשלב 3 — שמור בזיכרון השיחה.
