# 05 — קטגוריה 5: אוטומציות יומיות

המטרה: לקבוע אם המשתתף רוצה את האוטומציות המתוזמנות (אגנדת בוקר, סיכום ערב) ובאיזו שעה.

## למה חשוב

הבוט יכול לשלוח לה הודעות אוטומטיות בשעות קבועות:

- **אגנדת בוקר** — מושך פגישות מ-Google Calendar ושולח רשימה
- **סיכום ערב** — מושך מ-Airtable את הנתונים של היום (לידים, עסקאות, משימות)

זה לא חובה. חלק מהמשתתפות יעדיפו צ'אט יזום בלבד.

## השאלון

שאלי:

```
קטגוריה 5 מתוך 6: אוטומציות יומיות.

הבוט יכול לשלוח לך הודעות אוטומטיות בכל יום:

🌅 אגנדת בוקר — בשעה שתבחרי, הבוט שולח לך:
   "היי [שמך], בוקר טוב 😊
   סדר היום שלך:
   ✅ 09:00 פגישה עם...
   ✅ 11:00 פגישה עם...
   האם יש לך משימה בשבילי להיום?"

   * דורש חיבור ל-Google Calendar (נראה איך אחר כך)

🌙 סיכום ערב — בשעה שתבחרי, הבוט שולח:
   "📅 [תאריך]
   🆕 לידים חדשים: 3
   💰 סגירות היום: 1
   💵 הכנסות חודש: ₪18,000
   ✅ משימות שהושלמו: 5
   📋 משימות פתוחות: 2"

איזו אוטומציה את רוצה?

1. שתיהן (מומלץ)
2. רק אגנדת בוקר
3. רק סיכום ערב
4. אף אחת — אני אדבר עם הבוט יזומה
```

**שמור:**

```
WANT_MORNING_AGENDA = true / false
WANT_DAILY_SUMMARY  = true / false
```

---

## אם בחר אגנדת בוקר

שאלי:

```
מצוין. אגנדת בוקר —

1. באיזו שעה? (לדוגמה: 07:00, 08:00, 08:30)

2. חיבור ל-Google Calendar — קישור סודי iCal (2 דקות, בלי OAuth):
   - פתח https://calendar.google.com
   - גלגל שיניים → הגדרות
   - בתפריט שמאל: "הגדרות של היומנים שלי" → לחץ על השם של היומן שלך
   - גלול לתחתית עד "שילוב יומן"
   - העתק את "כתובת סודית בפורמט iCal" (Secret address in iCal format)
     ⚠️ לא את "כתובת ציבורית" — הסודית בלבד.
   - הכתובת מכילה /ical/ ו-/private- ומסתיימת ב-.ics

   רוצה לעשות את זה עכשיו או בשלב הפריסה?
```

**שמור:**

```
MORNING_AGENDA_HOUR     = "08:00"   (פורמט HH:MM)
GOOGLE_CALENDAR_ICAL_URL = (אם הביא עכשיו — לשמור; אם בפריסה — נסמן TODO)
```

> 📝 **למה iCal ולא OAuth?** ל-morning agenda צריך רק לקרוא מהיומן. iCal secret URL נותן בדיוק את זה (read-only, 2 דק' הקמה), בלי Google Cloud Console, בלי Consent Screen. לכתיבה ליומן (יצירת פגישות מצ'אט) הבוט המקומי משתמש ב-MCP דרך Claude — שלב נפרד שלא תלוי באוטומציות.

**אבטחה:** הקישור סודי. שמור ב-.env בלבד. אם דלף — בקלות מבטלים ויוצרים חדש מאותו עמוד (כפתור "אפס").

## אם בחר סיכום ערב

שאלי:

```
מעולה. סיכום ערב —

1. באיזו שעה? (לדוגמה: 20:00, 21:00, 22:00)
2. הסיכום מושך מטבלאות Airtable שכבר הגדרנו בקטגוריה 4.
   האם יש משהו ספציפי שאת רוצה שייכלל בסיכום שלא ציינתי?
   (לדוגמה: סטטיסטיקה על מקור הלידים, יחס המרה, וכו')
```

**שמור:**

```
DAILY_SUMMARY_HOUR = "20:00"
DAILY_SUMMARY_EXTRA = "כלום" / "פירוט לפי מקור" / וכו'
```

## טיפים לדיאלוג

- **אגנדת בוקר ב-08:00** מאוד מקובלת. אם הוא לא יודע — הצע את זה.
- **סיכום ערב ב-20:00** מתאים לרוב בעלי עסקים. אם הוא עובד מאוחר — 22:00 גם בסדר.
- **אם הוא רוצה שתי אוטומציות אבל לא Google Calendar** — הצע שאגנדת הבוקר תושך רק מ-Airtable (משימות) במקום Calendar. זה גמיש.
- **תזכורת על אזור זמן** — הבוט עובד באזור זמן `Asia/Jerusalem`. אם הוא בחו"ל באופן קבוע — נגדיר אחר.
