# talbs-bot-kit

סקיל להתקנת בוט וואטסאפ אישי עם Claude. מבוסס על בוט עבודה אמיתי של טל בשור.

## למשתתפי הסדנה של טל בשור

### התקנה (פקודה אחת)

```bash
cd ~/.claude/skills && git clone https://github.com/talbs1988-dotcom/talbs-bot-kit.git
```

או הורידי את [קובץ ה-.skill האחרון](https://github.com/talbs1988-dotcom/talbs-bot-kit/releases) ושימי בתיקיית `~/.claude/skills/`.

### הפעלה

ב-Claude Code, תכתבי:

> תקין לי בוט וואטסאפ

הסקיל ייכנס לפעולה ויעבור איתך 6 קטגוריות עד שיש לך בוט פעיל.

## מה הסקיל מכיל

- ✅ בוט שעובד עם Green API + Claude + Airtable
- ✅ מערכת חשבונית ירוקה (4 סוגי מסמכים, חסימת זיכוי)
- ✅ אוטומציות יומיות (אגנדת בוקר + סיכום ערב) — קוד פייתון עם API ישיר, יציב
- ✅ פריסה ב-Mac / Windows / ענן Railway
- ✅ חיבור MCPs מקומיים (Airtable + Google Calendar)

זמן התקנה משוער: 45-90 דק' לבוט בסיסי, 2 שעות לבוט מלא (כולל Calendar OAuth + חשבוניות).

## עלויות

- Green API (WhatsApp): ~$12/חודש
- Anthropic API או Claude Pro: $20-200/חודש (תלוי בשימוש)
- Railway (אם בענן): $5/חודש
- Airtable: חינם בתוכנית בסיסית

## רישיון

קוד פתוח. שימוש חופשי, ללא אחריות.
