# מדריך בנייה — הקוד, העיצוב והחוק

## בחירת סטאק

- **דף מכירה מלא עם אנימציות:** Next.js + React + Tailwind CSS + Framer Motion
- **דף פשוט** (הרשמה, וובינר, דף תודה): קובץ HTML סטטי אחד + פונקציית שרת קטנה לטופס
- כשמתלבטים, בוחרים בפשוט. דף סטטי שעולה מהר מנצח פרויקט מפואר שנתקע

## מיתוג

הכול נלקח מ-`brand-config.md`: צבעים, לוגו, תמונות. אין צבעים מוגדרים? מציעים ללקוח 2-3 כיווני פלטה לפי אופי העסק ונותנים לו לבחור לפני שבונים.

- **RTL מלא:** `dir="rtl" lang="he"` על ה-html
- גופן עברי נקי מ-Google Fonts (Rubik, Heebo, Assistant) — אחד לכל הדף
- אנימציות עדינות של הופעה בגלילה (fade + תזוזה קטנה), פעם אחת לכל סקציה, לא קרקס

## חוויית משתמש — חובה

### כפתורי CTA

- כפתור אמיתי (`<button>`) עם גלילה חלקה לטופס — לא קישור `#` (נשבר באייפון)
- אחרי הגלילה: פוקוס אוטומטי על השדה הראשון בטופס
- **מזהה `id="checkout"` אחד בלבד בכל הדף** — רק על הטופס עצמו

### וידאו המלצות

- הסרטון נפתח ומתנגן **בתוך הדף** (חלון קופץ עם נגן מוטמע), לא זורקים את הגולש ליוטיוב — גולש שיצא לא חוזר
- החלון נסגר ב-Esc, בלחיצה על הרקע ובכפתור X
- תמונות מקדימות נטענות בעצלתיים (`loading="lazy"`)

### טפסים

- מינימום שדות: שם + טלפון או מייל. כל שדה נוסף מוריד המרות
- שגיאה בשליחה? הליד לא הולך לאיבוד: שמירת גיבוי (קובץ / מייל) והודעה ברורה לגולש
- מסך "תודה" מוצג רק אחרי ששליחת הפרטים באמת הצליחה

### תמונות

- דחיסה לפני העלאה: ראשית עד 200KB, פנים-דף עד 100KB, קטנות עד 50KB
- טעינה מועדפת (priority) רק לתמונת ה-Hero, כל השאר בעצלתיים

## תאימות חוקית — בכל דף, בלי יוצא מהכלל

זה לא קישוט, זה חוק בישראל. דף שאוסף פרטים בלי זה חושף את בעל העסק לתביעות.

### 1. דפים משפטיים

שלושה קישורים ב-Footer, מהכתובות שב-brand-config: **תקנון**, **מדיניות פרטיות**, **הצהרת נגישות** (ואם רלוונטי גם מחיקת נתונים). נפתחים בטאב חדש. אין ללקוח דפים כאלה? מייצרים עבורו גרסה בסיסית ומציינים שכדאי שעורך דין יעבור עליה.

### 2. בנר קוקיז

- מופיע שנייה אחרי טעינת הדף (לא למי שכבר בחר)
- 3 קטגוריות: חיוניים (תמיד פעיל) / ניתוח שימוש / שיווק
- 3 כפתורים: דחיית הכול / התאמה אישית / אישור הכול
- הבחירה נשמרת בדפדפן, ואפשר לפתוח מחדש מקישור ב-Footer
- **קוד המעקב (פיקסל) נטען רק אחרי הסכמה לקטגוריית שיווק**

### 3. נגישות (תקן ישראלי 5568, רמה AA)

- כפתור נגישות בצבע כחול תקני `#2563EB` (לא בצבע המותג), עם ווידג'ט: הגדלת טקסט, ניגודיות, עצירת אנימציות
- דף `/accessibility` עם הצהרת נגישות מלאה
- כותרות בהיררכיה תקינה, טקסט חלופי (alt) לכל תמונה, פוקוס נראה על כל כפתור וקישור

### 4. checkbox תקנון בטופס

"אני מאשר/ת את התקנון ואת מדיניות הפרטיות" — כפתור השליחה נעול עד שמסמנים.

### 5. Footer

שורת קישורים: הצהרת נגישות · תקנון · מדיניות פרטיות · עדכון העדפות עוגיות · צור קשר

- שם העסק · כל הזכויות שמורות
