# הבוט שלי 💛

בוט WhatsApp אישי שמתחבר ל-Claude שלך. נבנה במיוחד למשתתפות הסדנה של טל בשור.

## איך מתחילים

לחיצה כפולה על `start.command` (Mac).

הדפדפן ייפתח עם הוראות התחברות. סורקים QR מהוואטסאפ ושולחים הודעה לעצמכם.

## פיצ'רים

- **Self-chat עובד מהקופסה** — אין צורך במספר טלפון שני
- **חיבור ל-Claude Code CLI** — אפס עלות API נוספת
- **UI מקומי** — http://127.0.0.1:7655 לכל ההגדרות

## קבצים

- `bot.js` — מנוע הבוט
- `index.html` — UI מקומי
- `config.json` — הגדרות (שם, מודל, system prompt)
- `auth/` — חיבור WhatsApp (לא משתפים!)

## בעיות נפוצות

- **לא נפתח דפדפן?** היכנסו ידנית ל-http://127.0.0.1:7655
- **לא מקבלים תשובה?** וודאו ש-`claude` רץ ב-Terminal — נסו `claude --version`
- **רוצים להחליף מספר?** לוחצים "🔄 התחבר מחדש" ב-UI

נבנה ✨ לסדנה של טל בשור.
