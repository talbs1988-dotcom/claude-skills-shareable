# 🎬 סקיל עריכת וידאו ל-Claude Code

קיבלת סקיל שהופך סרטונים ארוכים לקליפים אנכיים מוכנים לרשת: חיתוך לרגעים החזקים, כתוביות קריוקי בעברית (מילה מודגשת בזמן שהיא נאמרת), **זום שעוקב אחרי הפנים שלך**, טיזרים עם מוזיקה, צנזור מילים שפייסבוק לא אוהב, וניקוי רעש רקע.

## מה צריך

- **Mac עם שבב Apple Silicon** (M1/M2/M3 ומעלה). התמלול משתמש ב-`mlx` של אפל.
- **Claude Code** מותקן.
- `Node.js` ו-`python3` (בדרך כלל כבר קיימים במק).

## התקנה (העתק-הדבק אחד)

לשמור את הקובץ `video-editing-skill.zip` בתיקיית ההורדות, לפתוח את הטרמינל (Terminal), ולהדביק:

```bash
mkdir -p ~/.claude/skills && unzip -o ~/Downloads/video-editing-skill.zip -d ~/.claude/skills
```

זהו. Claude Code יטען את הסקיל אוטומטית בשיחה הבאה.

## איך משתמשים

פשוט לבקש מ-Claude Code בשפה חופשית, למשל:

> "יש לי סרטון על שולחן העבודה, תחתוך לי אותו לקליפים עם כתוביות וזום שעוקב אחריי"

הסקיל ייטען לבד. בפעם הראשונה הוא יריץ `bash scripts/setup.sh` שמתקין את הכלים (`ffmpeg`, `mlx-whisper`, `opencv`) ומעתיק את הפונט מספריית הפונטים של המק שלך - זה קורה פעם אחת ולוקח כמה דקות.

## מה יש בפנים

- `SKILL.md` - כל התהליך, הפרמטרים, והטריקים.
- `scripts/` - סקריפטים מוכנים (מעקב פנים, חיתוך-זום, כתוביות, טיזרים, ניקוי רעש, ובקרת איכות אוטומטית לקליפים המוכנים).
- `models/` - מודלים לזיהוי פנים ולניקוי רעש, **מצורפים** (אין הורדות).
- `LICENSES.md` - הרישיונות והקרדיטים של כל הרכיבים.

## הערות

- הסקיל בנוי ל-**macOS Apple Silicon**. על מערכות אחרות חלק התמלול צריך התאמה.
- הפונט `Arial Bold` לא מופץ עם החבילה (מטעמי רישיון) - הוא מועתק אוטומטית מהמק שלך בהתקנה. אם בא לך פונט אחר, שמים קובץ `ttf` משלך ב-`fonts/`.
- רשימת המילים לצנזור נמצאת ב-`scripts/captions_hz.py` - אפשר להוסיף/להסיר לפי הצורך.
